Question: Which geometric tool is used to create angles with a specified measurement?

###
Answers:
(A) - Ruler

(B) - Protractor

(C) - Set square

(D) - Compass

###

(B) - nan

(C) - Thinks a set square can measure angles but is not designed to do so with precision like a protractor.

(D) - Misunderstands that a compass is used for drawing circles and does not measure angles.
